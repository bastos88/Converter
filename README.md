# 🜂 Conversor de Moedas – 

---

## 🔮 Propósito

Criar uma aplicação que consome uma API real de câmbio para converter moedas estrangeiras (Dólar, Euro e Libra) em Reais. Este projeto serve como uma etapa iniciática no domínio da integração de APIs e da lógica simbólica no Front-End.

---

## ⚙️ Funcionalidades

- ✅ Conversão de **USD, EUR e GBP para BRL** em tempo real
- ✅ Consumo de API pública `AwesomeAPI`
- ✅ Validação de input
- ✅ Interface limpa
- ✅ Modularização com ES6 (`import/export`)
- ✅ Estrutura de diretórios

## 🌐 API Utilizada

[AwesomeAPI - Cotação de Moedas](https://docs.awesomeapi.com.br/api-de-moedas)

Endpoint utilizado:
```bash

https://economia.awesomeapi.com.br/json/last/USD-BRL,EUR-BRL,GBP-BRL
```
